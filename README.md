
# 🧠 Inventory Agent

This is a simple AI-powered **Inventory Management Assistant** built using the **OpenAI Agent SDK** in Python.

It can:
- 📦 List inventory items
- ➕ Add new items
- 🔍 Find items by name

---

## 🛠️ Requirements

- Python 3.9+
- OpenAI Agent SDK

Install dependencies:

```bash
pip install -r requirements.txt
